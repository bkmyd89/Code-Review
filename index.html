WEEK ONE

Week One minimally used HTML and focused mostly on learning how to work with Github and it's repositories. We incorporated inline and blockline elements as well as some basic Markdown functions. The sites linked will have many of these simple attributes.

WEEK TWO

Week Two dives into using CSS stylesheets along side HTML and also helps teach about formatting our code. You'll notice my code starts to become cleaner and should also begin to integrate Read Me files. Webgage styling starts to blossom after we discuss divs, spans, box models and floats.

WEEK THREE

Week Three really starts to bring things together as we introduce Bootstrap stylesheets into the code and with the addition of the grid systems, formatting our first personal portfolio website becomes a little easier.
